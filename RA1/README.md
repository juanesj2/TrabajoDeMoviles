# App de Consola para Evaluaciones de Programación Móvil

Este repositorio contiene una aplicación de consola diseñada para ser utilizada como herramienta de evaluación en el curso de Programación para Dispositivos Móviles.

## Descripción General

La aplicación de consola es el primer tema de evaluación y tiene como objetivo familiarizar a los estudiantes con los conceptos básicos de la programación en un entorno de consola, sentando las bases para el desarrollo de aplicaciones móviles más complejas.

## Características Principales

*   **Interacción por Consola:** La aplicación se ejecuta y se interactúa completamente a través de la línea de comandos.
*   **Ejercicios de Lógica:** Incluye una serie de ejercicios que ponen a prueba la lógica de programación de los estudiantes.
*   **Manejo de Datos Básicos:** Demuestra el uso de variables, tipos de datos y estructuras de control fundamentales.
*   **Validación de Entrada:** Implementa mecanismos básicos para validar la entrada del usuario.
